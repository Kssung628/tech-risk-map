Notion 資料庫設定（一次做完）

你要確保每一筆都有唯一的 Node Key，並把該頁設定為可公開。

資料庫欄位要有
Name（Title） / Node Key（Text） / Layer / Category / Jurisdiction / Type / Taiwan Status
公開方式
資料庫頁 Share → Share to web 開啟；每筆頁面仍建議確認可開啟。
填連結的地方
下方的 NOTION_LINKS，把 Node Key 對應到該筆 Notion page 公開連結。
快速檢查
